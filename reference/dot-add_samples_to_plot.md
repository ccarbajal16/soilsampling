# Add Samples to Plot (Internal)

Add Samples to Plot (Internal)

## Usage

``` r
.add_samples_to_plot(p, samples, color, alpha, size)
```

## Arguments

- p:

  A ggplot object

- samples:

  An ss_samples object

- color:

  Point color

- alpha:

  Point transparency

- size:

  Point size

## Value

Modified ggplot object
