# Squared Distance Matrix for Lat/Lon Coordinates

Squared Distance Matrix for Lat/Lon Coordinates

## Usage

``` r
.haversine_distance_matrix(X, Y)
```

## Arguments

- X:

  A matrix of coordinates (n x 2), columns are lon, lat

- Y:

  A matrix of coordinates (m x 2), columns are lon, lat

## Value

A matrix of squared distances (n x m) in km^2
