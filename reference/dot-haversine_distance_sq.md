# Compute Squared Haversine Distance

Computes squared great circle distance for lat/lon coordinates.

## Usage

``` r
.haversine_distance_sq(lon1, lat1, lon2, lat2)
```

## Arguments

- lon1, lat1:

  Longitude and latitude of first point(s) in degrees

- lon2, lat2:

  Longitude and latitude of second point(s) in degrees

## Value

Squared distance in km^2 (approximate)
