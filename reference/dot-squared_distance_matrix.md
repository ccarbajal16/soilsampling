# Squared Euclidean Distance Matrix

Computes the squared Euclidean distance between each row of X and each
row of Y.

## Usage

``` r
.squared_distance_matrix(X, Y)
```

## Arguments

- X:

  A matrix of coordinates (n x 2)

- Y:

  A matrix of coordinates (m x 2)

## Value

A matrix of squared distances (n x m)
